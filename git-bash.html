Enviando arquivo para o Git Hub através do Git Bash

  1. git add "nomedoarquivo".

    Para adicionar todos os arquivos da pasta de uma só vez, basta digitar: "git add ." (ao invés do nome do arquivo, adicionar o ponto.)

  2. git commit -m "adicionado o arquivo git-bash.html" (mensagem de commit).

  3. git push -u / git push -u origin main
